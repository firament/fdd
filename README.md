# fdd
Automation scripts to bring a raw ubuntu installation to a known state

---

## Before Running

1. In fdd-data.sh
	- Line 10
	- Change `/full/path/to/this/file` to 'Actual path'
	- Do **NOT** include the trailing slash `/`
2. In setup-fdd.sh
	- Line 19
	- Change `<plain-text-password>` to 'Actual password'
